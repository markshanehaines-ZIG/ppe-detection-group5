## How to Run

1. Click the notebook file above
2. Click the **"Open in Colab"** badge at the top of the notebook
3. Set runtime to **GPU** (Runtime → Change runtime type → T4 GPU)
4. **Runtime → Restart runtime and run all**
5. Expected runtime: ~25–35 minutes

> ⚠️ The notebook is designed to run entirely in Google Colab with no local setup required. All dependencies are installed in the first cell and the dataset is downloaded automatically.
